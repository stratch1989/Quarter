# Кнопки главного экрана

## Верхняя панель

| Кнопка | Описание |
|---|---|
| История | Открывает экран истории операций |
| Стрик (🔥) | Показывает серию дней без перерасхода |
| Подписки (↻) | Открывает меню регулярных списаний |
| Настройки (⚙) | Открывает меню настроек |

## Меню настроек (попап)

| Пункт | Описание |
|---|---|
| 💰 Укажите бюджет | Ввод общей суммы бюджета |
| 📅 Дата | Выбор конечной даты бюджета |
| ℹ️ О проекте | Информация о приложении |
| 👤 Аккаунт | Вход / управление аккаунтом |

## Меню подписок (попап)

| Пункт | Описание |
|---|---|
| ✕ Отключить | Выключить активную подписку |
| 📅 Ежедневная | Списание каждый день |
| 📆 Еженедельная | Списание каждую неделю |
| 🗓️ Месячная | Списание каждый месяц |
| ⚙️ Свой интервал | Списание через N дней |
| 📋 Текущие подписки | Список активных подписок |

## Нампад

| Кнопка | Описание |
|---|---|
| 0–9 | Ввод суммы |
| . | Десятичный разделитель |
| ⁺∕- | Переключение расход/доход |
| ⌫ (Delete) | Удалить последний символ |
| ↶ (Undo) | Отменить последнюю операцию |
| ↵ (Enter) | Подтвердить трату/доход |

## Категории

| Кнопка | Описание |
|---|---|
| + | Открыть пикер эмодзи для добавления категории |
| Эмодзи-чипы | Выбрать категорию для траты |
